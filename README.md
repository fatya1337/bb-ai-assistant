
# Bug Bounty AI Assistant

This repository contains a Codex agent configuration designed to help with bug bounty reasoning and security research.

The assistant helps analyze:

- HTTP requests
- API endpoints
- recon results
- application behavior

It suggests logical test cases and helps researchers reason about possible security weaknesses.

## Structure

AGENTS.md — AI behavior instructions  
PROMPT.md — prompt template  
setup.sh — optional environment setup  

## Usage

Connect this repository to Codex.

Provide observations such as:

- HTTP requests
- endpoint lists
- unusual application behavior

The assistant will help analyze the logic and suggest things worth testing.
